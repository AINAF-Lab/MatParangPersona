# Penjana Prompt AI

Repositori ini menyediakan alat web mudah untuk menjana prompt AI berkualiti tinggi menggunakan kerangka **SCOFFE** dan persona AI. Sesuai untuk pelajar, guru, dan peminat AI yang ingin belajar literasi AI dengan cepat.  

Terdapat tiga halaman:  
- **Index** – Pengenalan & navigasi.  
- **General Prompt** – Cipta prompt fleksibel berdasarkan SCOFFE.  
- **Persona Prompt** – Cipta prompt berdasarkan persona AI.  

Semua fungsi dijalankan di pelayar, percuma, dan mudah digunakan. Selamat menjana prompt dan belajar AI!
