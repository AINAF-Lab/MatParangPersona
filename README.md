## üß≠ What is this?

The **Mat Parang Persona (MPP)** is an advanced AI persona framework designed to transform AI from a tool into a **strategic collaborative partner**. It is the reference implementation of the **A.I. Network Architecture Framework (AINAF)**.

This project provides ready-to-use prompts and a complete methodology for creating AI agents that are:
* **Human-Centric:** Designed around the user's knowledge, goals, and context.
* **Culturally-Grounded:** Informed by Southeast Asian narrative traditions and the philosophical `Tulang Empat Kerat` (Four-Boned Framework).
* **Architecturally Sound:** Built on a clear level-based system, from simple assistants to autonomous orchestrators.

## üí° The PUAP Method

At the core of AINAF is the **PUAP formula**:

**`PUAP = Problem + User + AI + Persona`**

### The Four Components

1. **P - PROBLEM:** The high-value challenge.  
   *Ask:* "What exactly is the problem? What does 'solved' look like?"  

2. **U - USER:** The human context.  
   *Ask:* "Who is the user? Expertise? Knowledge gaps? Working style?"  

3. **AI - ARTIFICIAL INTELLIGENCE:** The tool you select.  
   *Ask:* "Which AI model? Capabilities? Budget?"  

4. **P - PERSONA:** The specialized identity of the AI.  
   *Ask:* "Prime directive? Core principles? Knowledge & tools? Tone & style?"

### PUAP Workflow

1. **DIAGNOSE:** Define **P** and **U**.  
2. **DESIGN:** Select **AI** and forge **Persona**.  
3. **DEPLOY:** Assemble the system (prompt or full app).  
4. **ITERATE:** Refine persona based on feedback.

## üåè The Empat Kerat Philosophy

The **`Tulang Empat Kerat`** (Four-Boned Framework) ensures MPP is **wise, respectful, and grounded**:

1. **Hati (Heart):** Emotional & ethical intelligence.  
2. **Fikiran (Mind):** Logical & structural intelligence.  
3. **Semangat (Spirit):** Purpose & motivation.  
4. **Prinsip (Principles):** Moral codex & epistemic humility.

## üîù Persona Levels

| Level | Name | Core Function | Example |
|-------|------|---------------|---------|
| 1     | Role-Player | General conversation | "Pirate advisor" |
| 2     | Domain Expert | Knowledge retrieval | JPJ chatbot |
| 3     | Synthesis Partner | Multi-step reasoning & creation | Mekanik Tua, Pencari Sage |
| 4     | Orchestrator | Manages team of Level 3 agents | Mat Parang Persona (MPP) |
| 5     | Autonomous Agent | Self-directed research | Prompt generator creating L1-L4 prompts |

**DIY Ceiling vs App Floor**:  
*Levels 1‚Äì3* can be run in free AI chat.  
*Level 4* requires an app managing multiple agents.  
*Level 5* is a pre-autonomous prompt creator.
